Installation

The NMR Sample Manager can be accessed online or downloaded for offline use. In both cases, all data and metadata stays on your computer - there is no server and nothing is shared or uploaded.

You can use the application in two ways:

  • Online: Access directly at https://nmr-samples.github.io/online/
  • Offline: Download and run locally on your computer

Optional TopSpin integration scripts are available for both online and offline usage.

Browser Requirements

You need Chrome, Chromium, or Edge for full functionality. Other browsers are not supported.

Online Access

Simply navigate to https://nmr-samples.github.io/online/ in your browser. When prompted, grant file system permissions to access your NMR data directories.

Offline Access

For offline use or if you prefer to run the application locally:

  1. Download the application: nmr-samples.zip
  2. Extract the zip file to a convenient location on your computer
  3. Open index.html in Chrome or Edge
  4. Grant file system permissions when prompted
  5. Set your NMR data root directory and start managing samples

Setting Up Directories

Once the application is running:

  1. Set Root Directory: Click "Set" next to "Root:" and select your main NMR data folder (e.g., /data/nmr or C:\TopSpin\data)
  2. Browse to Experiment: Click "Browse" to navigate to a specific experiment folder
  3. Ready to go: You can now create and manage samples in this folder
