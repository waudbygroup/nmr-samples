Installation Guide

Quick Start

The NMR Sample Manager is a web application that runs entirely in your browser - no installation required!

  1. Download the application files to your computer
  2. Open index.html in Chrome or Edge
  3. Grant file system permissions when prompted
  4. Set your NMR data root directory
  5. Start managing samples

Browser Requirements

Recommended: Chrome 86+ or Edge 86+ for full functionality including direct file access.

Limited support: Firefox and Safari work with manual file selection only.

Not supported: Internet Explorer or very old browsers.

File System Permissions

When you first run the application, Chrome/Edge will ask for permission to access your files. This is required to:

  • Read existing sample metadata files
  • Save new sample information
  • Browse your NMR experiment directories

Click "Allow" when prompted. You can revoke these permissions anytime in your browser settings.

Directory Setup

Set your root directory to point to your main NMR data folder (e.g., /data/nmr or C:\TopSpin\data). The application will then let you browse to specific experiment folders within this root.

TopSpin Integration (Optional)

For seamless TopSpin workflow integration, Python scripts will be available soon that automatically open the sample manager when you inject or eject samples. These scripts will use the existing ij and ej commands but add sample logging functionality.
