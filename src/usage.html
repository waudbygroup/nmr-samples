Usage Guide

=€ Quick Start

  1. Set Root Directory: Click "Set" next to "Root:" and select your NMR data folder
  2. Browse Experiments: Click "Browse" to navigate to specific experiment folders
  3. Manage Samples: Use "New Sample", "Duplicate", "Edit", or "Eject" buttons
  4. View Timeline: Click "Show timeline" to see complete experiment history

=Ë Sample Management

Creating Samples

  • New Sample: Create a fresh sample entry from scratch
  • Duplicate: Copy an existing sample for variations (titrations, pH series)

Managing Samples

  • Edit: Modify sample information and metadata
  • Eject: Mark sample as ejected with automatic timestamp

Sample Information

  • Users: Track who prepared/measured the sample
  • Sample: Protein/molecule details, concentrations, labelling
  • Buffer: pH, components, chemical shift references
  • NMR Tube: Tube type, volume, SampleJet positions
  • Lab References: Notebook entries, experiment IDs

=Ê Timeline View

The timeline provides a complete view of your experiment history:

  • Color-coded sessions: Each sample gets a unique color
  • Injection/ejection events: Track sample lifecycle
  • Sample details: Quick access to sample metadata
  • Chronological ordering: See experiment progression over time

=¡ Tips

Best Practices

  • Set a clear root directory structure
  • Use descriptive sample labels
  • Always fill in Users field
  • Include buffer pH when relevant
  • Record SampleJet positions

Keyboard Shortcuts

  • Ctrl+S: Save sample
  • Esc: Cancel editing

File Management

  • Samples saved as JSON files
  • Timestamped filenames
  • Human-readable format
  • Git-friendly storage

=' TopSpin Integration

The app supports URL parameters for TopSpin workflow integration:

index.html?folder=/path/to/experiment&action=inject

Actions supported:

  • inject: Launch with new sample form
  • eject: Automatically eject active sample
